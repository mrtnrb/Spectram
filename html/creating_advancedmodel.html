Section 1 Title

Models can be created 'manually'. Handles to anonymous functions must be provided in the form @(x,p1,p2,...pN)(fun). With x being the independent (control variable) and p1...pN are the parameter used in fitting. Constants should not be stored in the anonymous function because they will get lost. For keeping parameters fixed during fitting they should be stated directly, i.e.:

q = 3;
fh = @(x,p)(p.*x+q);

should be avoided. Instead, use:

fh = @(x,p)(p.*x+3)

Also, any Matlab function myfun with y = myfun(varargin) can be used as shown below exampled with polyval:

Model = {@(x,b,f) sin(x/b)+f ...
         @(x,a,b,c) polyval([b,c,a],x)...
         model_fun.invexpdec};

Evaluation of the model

For any model a parameter name vector and an array of zeros of the necessary format for model evaluation can be generated by the vecpar() function:

[Paranames, Parameters] = vecpar(Model);

The model can then be evaluated by:

x = 1:10;
value = eval_model(x, Model, Parameters);
